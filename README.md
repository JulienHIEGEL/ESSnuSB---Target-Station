# Overview  
  
This repository contains work related to the target station studies for the ESSnuSB+ project.  
The current focus is on pion+ and proton fluence simulations using the following tools and versions:  
  
`FLAIR` version: `3.3.1`  
`FLUKA` version: `4.4.1`
